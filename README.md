# 🦁 CJU LIKELION: Possibility to Reality

청주대학교 멋쟁이사자처럼(CJU LIKELION)은 기술을 통해 가능성을 현실로 만드는 실전형 IT 커뮤니티입니다.
우리는 단순히 코딩 기술을 익히는 것을 넘어, 기획부터 배포까지의 전 과정을 체계적으로 학습하며 협업의 가치를 실천합니다.

---

## 🎯 핵심 가치 및 목표

* **실전 지향적 학습**: 이론을 넘어 실제 서비스 기획부터 최종 배포까지 전체 제품 사이클을 직접 경험합니다.
* **현업 표준 협업**: Gitflow 전략, Pull Request 기반 Code Review, CI/CD 자동화 등 실제 IT 기업의 프로세스를 체계적으로 습득합니다.
* **기술적 성장**: Java/Spring, React, Figma 등 파트별 핵심 기술 스택을 마스터하여 경쟁력 있는 인재로 성장합니다.

---
*아이디어를 현실로 만드는 16주의 여정, CJU LIKELION 13기가 함께합니다.*
